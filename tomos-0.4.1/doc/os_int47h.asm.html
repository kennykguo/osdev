os_int47h_setup

Set TomOS system calls on int 47h. Cant be called by user.It is called by system during startup.

INPUT:

Nothing

OUTPUT:

Nothing
Section : tomos syscalls

see: os_main.asm

os_int47h

int47 handler.

INPUT:

AX - id of syscall

OUTPUT:

if ax out of bound return 0FFFFh
Section : tomos syscalls

os_int47h_get_syscall

Get addres of syscall.

INPUT:

AX - id of syscall

OUTPUT:

DX:DI - syscall address
if ax out of bound return 0FFFFh
Section : tomos syscalls

os_int47h_set_syscall

Set addres of syscall.

INPUT:

AX - id of syscall
DX:DI - syscall address

OUTPUT:

AX - 0FF if id out of bound
if ax out of bound return 0FFFFh
Section : tomos syscalls
