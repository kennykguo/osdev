Documentation for TomOS

Documentation for TomOS was generated by program docgen which is a part of TomOS since ver0.0.1. To genarate documentation execute ./build-doc.sh from TomOS directory. To add new comments or modify old ones see doc/docgen.txt.

Sections

All system calls are grouped into sections to ease a user moving around systemcalls. Every section indicates function of the system call. For example 'os_print_string' belongs to section 'string'.
